# Issue: Inventory System

## Priority: HIGH
## Status: OPEN

## Description
Implement a full Minecraft-style inventory system with 27 storage slots, 9 hotbar slots, 4 armor slots, and 1 off-hand slot. Items stack up to 64 (or 16 for special items like ender pearls).

## Requirements (from minecraft.wiki)
- Open/close inventory with E key
- 2x2 crafting grid in inventory (4 slots)
- 3x3 crafting grid when crafting table is placed
- Armor equipping in armor slots
- Item stacking (64, 16, or 1)
- Item dragging and dropping between slots
- Item splitting (right-click to place one)

## Acceptance Criteria
- [ ] Inventory opens with E key
- [ ] 27 storage slots visible
- [ ] 9 hotbar slots visible
- [ ] Items can be picked up and placed in slots
- [ ] Items stack correctly
- [ ] 2x2 crafting grid works
- [ ] Armor slots accept armor items
