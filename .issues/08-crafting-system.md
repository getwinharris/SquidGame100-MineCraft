# Issue: Crafting System

## Priority: HIGH
## Status: OPEN

## Description
Implement Minecraft crafting with shaped and shapeless recipes, crafting table 3x3 grid, and recipe book.

## Requirements (from minecraft.wiki)
- Shaped recipes (specific pattern in grid)
- Shapeless recipes (any arrangement)
- Crafting table block with 3x3 grid
- Recipe book for known recipes
- Output slot showing crafted item

### Core Recipes (from wiki)
- 1 Log -> 4 Planks
- 2 Planks -> 4 Sticks
- 3 Planks -> Crafting Table
- 1 Cobblestone + 2 Sticks -> Wooden Pickaxe
- 3 Cobblestone + 2 Sticks -> Stone Pickaxe
- 1 Iron Ingot + 2 Sticks -> Iron Pickaxe
- 1 Diamond + 2 Sticks -> Diamond Pickaxe
- 5 Iron Ingots -> Iron Chestplate
- 1 Obsidian + 2 Diamonds -> Enchanting Table
- 3 Iron Ingots + 1 Blaze Rod -> Brewing Stand

## Acceptance Criteria
- [ ] Crafting table block can be placed
- [ ] Right-click opens 3x3 crafting grid
- [ ] Shaped recipes work
- [ ] Shapeless recipes work
- [ ] Output slot shows result
- [ ] Recipe book shows available recipes
- [ ] Crafted items go to inventory
