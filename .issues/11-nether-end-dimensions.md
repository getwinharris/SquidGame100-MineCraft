# Issue: Nether and End Dimensions

## Priority: MEDIUM
## Status: OPEN

## Description
Implement the Nether and End dimensions with portal mechanics matching Minecraft Java Edition.

## Requirements (from minecraft.wiki)

### Nether
- Nether portal: Obsidian frame (minimum 4x5) lit with flint & steel
- Nether wastes, basalt deltas, crimson forest, warped forest, soul sand valley biomes
- Nether mobs: Zombified Piglin, Blaze, Ghast, Wither Skeleton, Magma Cube, Piglin, Hoglin
- Nether ores: Nether Quartz, Nether Gold, Ancient Debris
- Lava oceans, Soul Sand, Glowstone, Nether Wart

### The End
- End portal in Stronghold (12 Eyes of Ender required)
- Ender Dragon boss (720 HP)
- End Cities with Shulkers and Elytra
- Chorus Fruit plants

### Portal Mechanics
- Nether portal: 1 block in Nether = 8 blocks in Overworld
- End portal: Found in Stronghold
- Respawn anchors for Nether spawn points

## Acceptance Criteria
- [ ] Nether portal can be built and activated
- [ ] Teleporting to Nether works
- [ ] Nether terrain generates correctly
- [ ] Nether mobs spawn
- [ ] End portal can be found and activated
- [ ] Ender Dragon boss fight works
- [ ] End Cities generate
- [ ] Respawn anchors work in Nether
