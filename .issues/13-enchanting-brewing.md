# Issue: Enchanting and Brewing

## Priority: MEDIUM
## Status: OPEN

## Description
Implement enchanting table and brewing stand mechanics matching Minecraft Java Edition.

## Requirements (from minecraft.wiki)

### Enchanting
- Enchanting Table: requires Lapis Lazuli + XP levels
- Bookshelves increase max enchantment level (max 30)
- Random enchantments per level
- Enchantment types: Sharpness, Efficiency, Protection, Unbreaking, etc.
- Anvil: combine enchanted items, rename, repair

### Brewing
- Brewing Stand: Nether Wart + ingredients
- Water Bottle base
- Potion types: Healing, Strength, Speed, Fire Resistance, etc.
- Splash Potions: throwable
- Lingering Potions: area effect
- Ingredients: Blaze Powder, Ghast Tear, Magma Cream, etc.

## Acceptance Criteria
- [ ] Enchanting table can be crafted and placed
- [ ] Bookshelves increase enchantment level
- [ ] Enchanting costs XP and Lapis
- [ ] Enchantments apply to tools/armor
- [ ] Brewing stand can be crafted and placed
- [ ] Potions can be brewed
- [ ] Potions give status effects
- [ ] Splash potions work
- [ ] Anvil combines enchanted items
